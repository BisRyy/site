Here's a collection of the web projects I have been involved in over the past few years. Haven't been doing mobile in awhile now. >_<

Una

Una is a free and open source Javascript library that allows developers to create interactive applications where multiple users can interact with large central displays in real-time using mobile devices. This project was completed in the course of CS3216 Software Development on Evolving Platforms and won the Best Project Award for STePS (NUS School of Computing Term Project Showcase) 2013/14 Semester I.

Node.js Socket.io JavaScript HTML5 CSS3

Nutty Ninjas

Nutty Ninjas is a real-time shooter that brings the experience of social multiplayer gaming to a whole new level; it is a console-style game that can be played anywhere and anytime, simply with your computer and mobile devices. Multiple players can join a common gameplay screen just by using their mobile devices, and control their ninja character to unleash dangerous weapons at fellow players!

It is developed using the cross-display library Una. I played the role of UI/UX designer and front-end developer for this project.

Node.js Una JavaScript HTML5 CSS3 Box 2D

EasilyDo Corporate Site

EasilyDo's corporate site was solely developed and maintained by me while I was an interning at EasilyDo. I also developed the site's plugins to maintain a responsive layout. A Wordpress theme was created for the EasilyDo blog so that it was consistent with the other static pages of the site.

Wordpress JavaScript HTML5 CSS3

Corspedia

Corspedia is an easy-to-use search engine for National University of Singapore's CORS Bidding Archives. It provides a clean and simple interface for students to search for the bidding history of any module. Built using AngularJS and Google App Engine.

JavaScript Python HTML5 CSS3

Osmose

Osmose is a platform to facilitate the interaction and sharing between students and instructors of a course. Students can ask questions, answer questions from their peers, and also share new findings with one another.

Sails.js Node.js JavaScript HTML5 CSS3

#ifihadglass

My entry for Google's #ifihadglass competition where participants demonstrate potential uses for Google Glass and selected entries can be part of the Glass Explorer Program. I used jQuery and HTML5/CSS to simulate the Google glass interface to demonstrate potential use-cases of Google Glass. View the full site here.
